# 🌍 WanderTales

A full-stack MERN travel blogging platform that enables users to create, share, and explore travel experiences with rich media, location-based storytelling, and community engagement.

---

## 🚀 Features

- 🔐 Secure JWT Authentication
- 👤 Role-Based Access Control
- 📝 Create, Edit, and Delete Travel Blogs
- 📍 Location Tagging
- 🖼️ Image Upload Support
- ❤️ Like and Comment System
- 🌐 Public & Private Post Visibility
- 🔍 Browse and Discover Travel Stories
- 📱 Responsive User Interface
- ⚡ RESTful API Architecture

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/jennifer2006-github/WanderTales.git
```

### Navigate to Project

```bash
cd WanderTales
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
